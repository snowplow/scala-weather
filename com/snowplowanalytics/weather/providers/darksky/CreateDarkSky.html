    Create a DarkSkyCacheClient capable of caching results

    Create a DarkSkyCacheClient capable of caching results

    apiHost

    URL to the Dark Sky API endpoints

    apiKey

    API key from Dark Sky

    timeout

    time after which active request will be considered failed

    cacheSize

    amount of responses stored in the cache

    geoPrecision

    nth part of 1 to which latitude and longitude will be rounded stored in cache. e.g. coordinate 45.678 will be rounded to values 46.0, 45.5, 45.7, 45.78 by geoPrecision 1,2,10,100 respectively. geoPrecision 1 will give ~60km accuracy in the worst case; 2 ~30km etc

    returns

    either an InvalidConfigurationError or a DarkSkyCacheClient
